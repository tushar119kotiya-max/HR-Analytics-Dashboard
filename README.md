# HR Analytics Dashboard

## 📊 Project Overview
This project is an interactive HR Analytics Dashboard built using Microsoft Excel.

## 🛠 Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting

## 📈 KPIs
- Total Employees: 1470
- Active Employees: 1233
- Attrition Count: 237
- Attrition Rate: 16%
- Average Age: 37 Years

## 📌 Dashboard Features
- Employee Gender Analysis
- Department-wise Attrition
- Job Role-wise Attrition
- Age Group Analysis
- Marital Status Analysis
- Interactive Filters

## 🔍 Key Findings
- Overall Attrition Rate is **16%**.
- Total Employees are **1470**.
- Sales Executive and Research & Development show high attrition.
- Employees aged **26–35 years** have the highest attrition.
- Male employees are higher than female employees.
